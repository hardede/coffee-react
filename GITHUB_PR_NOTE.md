# Где смотреть PR

В этой среде агент не пушит напрямую в ваш удалённый GitHub-репозиторий через `git push`.
PR создаётся через интеграционный инструмент `make_pr`.

Если PR не отображается в вашем GitHub интерфейсе:
1. Проверьте, подключена ли интеграция репозитория к текущей среде/агенту.
2. Проверьте, что текущая ветка отслеживается удалённым репозиторием.
3. После подключения интеграции повторный вызов `make_pr` создаст/обновит PR с теми же коммитами.
